# 发散
风格：2D
类型：即时策略
难点：病毒与细胞的相关知识
游戏核心：作为病毒要侵入本地生物来进行繁殖和入侵，在侵入时能够不断地增加新的病毒，新的病毒可以用来侵入新的部位，也要同时抵制体内细胞的反侵蚀，最终通过侵入心脏来达到入侵整个动物，
类似参考：红警、废墟图书馆
开发难点：
● 入侵时候的细胞的反扑，细胞AI操作
● 病毒升级成什么，升几级，最终形态是什么
● 美术需要有超高脑洞
开发难点：✡✡✡

# 浮
风格：2D
类型：
难点：不同道具在
游戏核心：通过给予水中玩家不同的道具让他们能够更快的到达水面
类似参考：没想到
开发难点：
- 不同道具能够提供的上升区域和上升时间不同
- 不同NPC所需要的时间不同
- 在拯救数量达到一定程度后提供给玩家当前道具无限制使用（直到奖励时间结束）
开发难度：✡
# 入
风格：2D横板解密
类型：解密
难点：门与门之间的链接，在转变之后又转变回来，不同机关触发所导致的效果不同
游戏核心：通过进入不同的“门”来进入到不同的区域，并通过触发不同的机关来达到使“门”能够链接另一个门的效果，并最终走到平台对面
类似参考：
开发难点：
● 机关之间的触发有无关联，如果关联，要怎么样才能让它们关联起来，并达到改变门连接到另一个门
● 不同门之间的链接
● 触发机关后可能会使得门、平台、链接其中之一或者其中之二发生改变
开发难度：✡✡
